4. PUSH-TO-TALK SIMULATION
   The PTT (Push-To-Talk) sync feature simulates a standard Windows keyboard
   key press using the Windows SendInput API — the same mechanism used by
   accessibility software and macro keyboards. This is a normal user-level
   keyboard event, identical to physically pressing a key.

5. AUDIO ROUTING
   Audio is routed through standard Windows audio devices only. Integration
   with virtual audio drivers (e.g., VB-CABLE) requires those drivers to be
   installed separately by the user.
